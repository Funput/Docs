# Tổng quan cài đặt

Funput hỗ trợ **macOS**, **Windows** và **Linux**. Chọn hướng dẫn phù hợp với hệ điều hành bạn đang dùng:

| Nền tảng | Hướng dẫn | Trạng thái |
| -------- | --------- | ---------- |
| macOS | [Cài đặt trên macOS](./macos) | Sẵn sàng |
| Windows | [Cài đặt trên Windows](./windows) | Sẵn sàng |
| Linux | [Cài đặt trên Linux](./linux) | Sẵn sàng |

## Tải xuống

Tất cả bản cài đặt được phát hành trên GitHub:

- [Bản phát hành (GitHub Releases)](https://github.com/Funput/Funput/releases)

Mỗi bản phát hành kèm file checksum **SHA-256** (`.sha256`) để bạn kiểm tra tính toàn vẹn của file tải về.
