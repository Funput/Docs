# Cài đặt trên Linux

:::info Đang bổ sung
Nội dung hướng dẫn cài đặt chi tiết cho Linux đang được soạn thảo. Trong lúc chờ, bạn có thể tải bản phát hành mới nhất từ GitHub.
:::

## Tải xuống tạm thời

- [Bản phát hành (GitHub Releases)](https://github.com/Funput/Funput/releases)

Các gói Linux thường gồm `.deb` (Debian/Ubuntu) và `.rpm` (Fedora/openSUSE), cho cả **IBus** và **Fcitx5**.
