# Cài đặt trên Windows

:::info Đang bổ sung
Nội dung hướng dẫn cài đặt chi tiết cho Windows đang được soạn thảo. Trong lúc chờ, bạn có thể tải bản phát hành mới nhất từ GitHub.
:::

## Tải xuống tạm thời

- [Bản phát hành (GitHub Releases)](https://github.com/Funput/Funput/releases)

Trên Windows, Funput phát hành dưới dạng file **`Funput-*.exe`** portable.
