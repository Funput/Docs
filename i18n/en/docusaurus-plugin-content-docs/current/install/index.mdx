# Installation overview

Funput supports **macOS**, **Windows**, and **Linux**. Choose the guide for your operating system:

| Platform | Guide | Status |
| -------- | ----- | ------ |
| macOS | [Install on macOS](./macos) | Available |
| Windows | [Install on Windows](./windows) | Coming soon |
| Linux | [Install on Linux](./linux) | Coming soon |

## Download

All installers are published on GitHub:

- [Releases (GitHub)](https://github.com/Funput/Funput/releases)

Each release includes a **SHA-256** checksum file (`.sha256`) so you can verify your download.
